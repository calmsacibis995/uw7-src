3Com 59X series PCI adapters

SCO Gemini systems support the following 3Com59X PCI adapters:

  • 3Com 590 (10Mb)

  • 3Com 595 (10/100Mb)

Begin the configuration process by running your machine's PCI configuration routine. After configuring your adapter, run the Network Configuration Manager to configure your network driver.
