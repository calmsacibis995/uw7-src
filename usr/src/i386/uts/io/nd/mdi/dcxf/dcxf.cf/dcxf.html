Digital Fast EtherWORKS PCI 10/100 adapters

SCO Gemini systems support the following Digital Fast EtherWORKS PCI adapters:

  • DE500-XA

  • DE500-AA

  • DE500-BA

  • DE500-FA

To complete configuration for these adapters, the Network Configuration Manager prompts you for:

  • Media Type
    "Media Type" allows you to specify the specific speed and duplex setting. Choosing "Auto" enables the adapter to automatically negotiate these settings.

  • Store and Forward Mode
    (Advanced)Determines whether the adapter will be set to Store and Forward mode.

  • PCI Burst Size
    (Advanced)This may be set to 0, 1, 2, 4, 8, 16, or 32.

    NOTE:

    • The only acceptable settings for the DE500-FA are HDX_100 and FDX_100, for 100Mb, half-duplex and 100Mb, full-duplex. If any other option is selected, the adapter will default to HDX_100.
