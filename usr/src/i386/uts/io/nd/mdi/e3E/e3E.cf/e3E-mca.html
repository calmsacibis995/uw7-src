3Com 5X9 series MCA adapters

SCO Gemini systems support the following 3Com5X9 MCA adapters:

  • 3Com 529

Begin by installing your adapter and running your machine's MCA reference disk. After configuring your adapter, run the Network Configuration Manager to configure your adapter's driver; the Network Configuration Manager prompts you for the ``Slot number''. For more information, see ``Configuration parameters''.
