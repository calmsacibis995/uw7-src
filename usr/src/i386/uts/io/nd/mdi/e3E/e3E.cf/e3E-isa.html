3Com 5X9 series ISA adapters

SCO Gemini systems support the following 3C5X9 ISA adapters:

  • 3Com 509

  • 3Com 509B

To complete configuration for these adapters, the Network Configuration Manager prompts you for:

  • Interrupt Vector

  • I/O Base Address

  • Cable Type
    When Cable Type is set to AUTO, the transceiver type that will be used to send and receive packets is automatically determined by the software driver. A cable should be physically connected to the adapter for this type of automatic configuration. However, if there is more than one cable connected to the adapter, the order that will be used is 1) internal TP connector, 2) External AUI connector and 3) internal BNC connector.
For more information, see ``Configuration parameters''.
