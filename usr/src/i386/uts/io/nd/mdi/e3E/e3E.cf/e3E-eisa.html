3Com 5X9 series EISA adapters

SCO Gemini systems support the following 3Com5X9 EISA adapter:

  • 3Com 579

Begin the configuration process by running your machine's EISA configuration routine. Next, configure your system, then run the Network Configuration Manager to configure your network driver.
