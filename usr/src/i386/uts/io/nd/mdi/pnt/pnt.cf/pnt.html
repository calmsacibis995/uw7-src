AMD PCNet PCI adapters

SCO Gemini systems support the following AMD PCNet PCI adapters:

  • PCNet-PCI
  • PCNet-PCI II
  • PCNet-FAST

Run the Network Configuration Manager to configure your networking driver and protocols.
